Requisitos:
	Conta no GitHub.
	Eclipse Java Neon.
Como baixar:
 	1) Clicar em Clone or Download (Botão verde)
	2) Copiar o link
	3) Criar uma pasta em um local de fácil acesso no seu computador
	4) Abrir o terminal na pasta criada no passo 3 e digitar git clone [link copiado no passo 2]
	5) Abrir o Eclipse Java Neon
	6) Clicar em File>Import>General>Projects from Folder or Archive
        7) Irá abrir uma janela e no campo Import Source clicar em directory e deve procurar pela pasta em que o projeto foi clonado.
	8) Clicar em Finish
	9) Na lateral, caminhar até a classe Java chamada MetodoCongruencia.java
	10) Procurar no menu acima o botão verde de executar.
	11) Informe a quantidade de números que deseja gerar.
